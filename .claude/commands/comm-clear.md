# /comm-clear

Setze Claude-Codex-MCP-Kommunikationsdateien sicher zurück.

1. Kläre den Projektordner.
2. Führe lokal aus:

```bash
npm --silent --prefix PROJEKT/WORKSPACE run comm -- clear --project-dir /pfad/zum/projekt
```

3. Nenne nur Backup-Ordner, zurückgesetzte Dateien und nächsten Schritt.

Keine Inhalte aus alten Dateien oder Backups ausgeben.
