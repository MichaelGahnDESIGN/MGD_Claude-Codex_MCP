# /comm-clear-backup

Räume alte Claude-Codex-MCP-Backups auf.

1. Kläre den Projektordner.
2. Führe lokal aus:

```bash
npm --silent --prefix PROJEKT/WORKSPACE run comm -- clear-backup --project-dir /pfad/zum/projekt
```

3. Melde, wie viele alte Backups gelöscht wurden und welches Backup bleibt.

Keine Inhalte aus Backups ausgeben.
