# Claude-Startpunkt

Dieser Ordner ist der lokale Startbereich für Claude Code und Claude Cowork.

Wichtig:

- Die verbindliche Root-Anweisung für Claude bleibt `CLAUDE.md`.
- Die gemeinsame Agentenlogik steht in `AI/AGENTEN/AGENTS.md`.
- Die Projektvorlage ist tool-unabhängig. Claude-spezifische Dateien sollen nur
  den Einstieg erleichtern und keine Regeln duplizieren.

Empfohlener Startprompt:

```text
Lies CLAUDE.md und readmefirst.md. Richte dieses Verzeichnis als saubere
Projektkopie ein und dokumentiere alle Entscheidungen auf Deutsch.
```
