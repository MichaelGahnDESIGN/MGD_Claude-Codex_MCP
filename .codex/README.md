# Codex-Startpunkt

Dieser Ordner enthält lokale Codex-Hilfen für diese Vorlage.

Wichtig:

- Dauerhafte Arbeitsregeln stehen im Root in `AGENTS.md`.
- Ausführliche Projektlogik steht in `AI/AGENTEN/AGENTS.md`.
- Wiederverwendbare Codex-Skills liegen zusätzlich unter `.agents/skills/`,
  weil Codex diesen Ort offiziell als repo-sichtbaren Skill-Pfad erkennt.
- Diese Datei enthält keine Secrets und keine projektspezifischen Zugangsdaten.

Empfohlener Startprompt:

```text
Lies AGENTS.md und readmefirst.md. Richte dieses Verzeichnis als neues Projekt
aus der Vorlage ein und halte OPENROUTER separat.
```
